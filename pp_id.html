Kebijakan Privasi "WaterStamp"

Motoki Takahashi (selanjutnya disebut "kami") berusaha melindungi informasi pengguna agar dapat menyediakan layanan yang terkait dengan aplikasi smartphone "WaterStamp" (selanjutnya disebut "aplikasi ini") dengan baik.

2. Pengumpulan Informasi Pengguna melalui Modul Pengumpulan Informasi
Aplikasi ini tidak menggunakan modul pengumpulan informasi.


3. Pengungkapan Informasi Pengguna kepada Pihak Ketiga
Dalam aplikasi ini, kami tidak mengirim informasi ke server pengembang, sehingga kami tidak mengungkapkan atau menyediakan informasi kepada pihak ketiga.

4. Publikasi Kebijakan Privasi
Kebijakan privasi dipublikasikan di layar "Kebijakan Privasi" dari "Layar Edit Pengguna" aplikasi ini dan di URL (https://).

5. Perubahan Kebijakan Privasi
Kami dapat mengubah kebijakan privasi ini karena perubahan dalam hukum dan peraturan.
